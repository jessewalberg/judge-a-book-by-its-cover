## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jessewalberg/judge-a-book-by-its-cover.git
   ```

2. The app requires a seeded database. You can check the prisma/migrations folder for creating the initial tables 

3. To seed this database I put the golang project that I used to do this in the scraper directory.

4. This golang project can be used as reference for your own scraper or you can use it to seed your own database. This is where you would need to use the google books api key

5. Install NPM packages
   ```sh
   npm install
   ```
6. Requires a DATABASE_URL environment variable

7. Run the program in development mode
    ```sh
    npm run dev
    ```
